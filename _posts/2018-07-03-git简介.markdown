# git命令记录

## 提交：
- 第一步：更改之后，切换到该目录下，输入shell：git add .
- 第二步：提交commit：git commit -m 我更改了xxx
- 第三步：push：git push xxx（reportery ssh 地址）
----
## 分支：
- 切换到master：git checkout master
- git pull origin master 
- git push xxx(reportery ssh 地址)
